Virtual nodes simulator. Each server is placed at V positions on the hash ring. A slider controls V; raising it makes the per-server load more even. Click a server to remove it and watch its keys scatter across the survivors.

How to read this Each server has V positions on the ring, called vnodes. With V=3, server alpha has a1, a2, a3 — each at a different hash, scattered around the ring. Same colour means same server. Slide V to add more, or hit the demo.
