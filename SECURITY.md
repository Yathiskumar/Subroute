# Security Policy

## Supported Versions

As Subroute is in early (Phase 1) development, security fixes are applied to the latest `main` branch.

| Version | Supported |
| ------- | --------- |
| `main`  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately through [GitHub's security advisory feature](https://github.com/Yathiskumar/Subroute/security/advisories/new). Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix, if known

We will acknowledge your report as quickly as possible and keep you informed as we work on a fix. Thank you for helping keep Subroute and its users safe.
